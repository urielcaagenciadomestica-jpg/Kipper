⚙️ Configuración inicial
Para conectar la app con Google Drive necesitas un Client ID de Google. Solo se hace una vez.
Paso 1
Crear proyecto en Google Cloud
Ve a console.cloud.google.com → crea un proyecto nuevo → activa la API de Google Drive y la API de Google Sheets.
Paso 2
Crear credenciales OAuth
En "Credenciales" → "Crear credenciales" → "ID de cliente OAuth" → tipo: Aplicación web. En "Orígenes autorizados" agrega la URL donde estará esta app (ej: https://tusitio.com o http://localhost para pruebas).
Paso 3
Pega tu Client ID aquí
Copia el Client ID (termina en .apps.googleusercontent.com) y pégalo abajo.
